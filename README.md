# NullClass Internship Projects

This repository contains all my internship projects completed during the NullClass Internship program. Each project focuses on different aspects of language translation and natural language processing.

## Projects

### 1. French-to-Tamil
A machine translation application that translates text from French to Tamil language.

### 2. Dual-Language-Translation
A bidirectional translation system supporting translation between two languages.

### 3. Translation-from-Image
An OCR-based translation system that extracts text from images and translates it.

### 4. Voice-Translator
A voice-enabled translator that converts spoken language to translated text or speech.

### 5. Logic-Building-task
A word translation application with specialized logic for handling vowels and time-based translation rules using machine learning.

### 6. Realtime-Conversation-with-Voice-Translation
A real-time conversation translator with voice input/output capabilities.

## Technologies Used

- Python
- Machine Learning (Seq2Seq, Transformers)
- Flask/Streamlit for Web Applications
- Speech Recognition & Text-to-Speech
- OCR (Optical Character Recognition)
- Natural Language Processing
